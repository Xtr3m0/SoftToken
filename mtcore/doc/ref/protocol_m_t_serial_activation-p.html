Detailed Description

This is the protocol implemented by the object that handles the serial and activation values.

On the Gemalto Ezio OATH Client API for iOS, the serial/activation pair is used to identify a token that must be deployed into the mobile token. This pair is send to the user by the token issuer.

Instances of this protocol can be acquired using the method MTCoreFactory::createSerialActivation:.

Note:
This library does not support off-line token deployment, thus this object will consider all off-line serial/activation pairs as invalid values.
