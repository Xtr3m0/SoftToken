Detailed Description

This static class implements the object factory used to create the instances of the main classes of the Gemalto Ezio OATH Client API for iOS.

Note:
Do not create instances of this class.

id <MTAppCore> MTCoreFactory::createCore:usingDatabase: ( NSString *  deviceId,
[usingDatabase] NSString *  dataDir 
) [static, virtual]

Creates a new instance of the application core for the specified device using the specified database.

Parameters:
[in]deviceIdA string with that represents a unique identifier for the device. This value must be the same all the times or the database will not work well.
[in]dataDirThe directory that will be used by this core as the database.
Returns:
The core instance or nil for failure.
Note:
The new instance is added to the autorelease pool, given that, do not forget to retain this instance if required.
Warning:
Do not instantiate more than one core using the same database at same time.

BOOL MTCoreFactory::resetDatabase: ( NSString *  dataDir) [static, virtual]

Deletes the database located on the given path.

This method may be used to reset the application when the database is dead or corrupted.

Parameters:
[in]dataDirThe path to the token database.
Returns:
YES for success or NO otherwise.
Warning:
Do not invoke this method if an instance of the MTAppCore is using this database.
